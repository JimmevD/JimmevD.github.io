Level Design (Personal Learning)

About the Project

Concept

This is a personal learning game about Level Design. I wanted to sketch, block-out and set dress 3 levels; One easy, medium and hard. The game uses the fab.com assets: "Ultimate FPS Puzzle Kit" as a base and "Brutalist Architecture Office" for most of the assets.

Details

Engine: Unreal Engine
Genre: First person combat
Platform: PC (Windows)
