Combat System

About the Project

Concept

This combat system includes a melee sword attack, shield bash, ranged bow attack, and magic fireball. You can use these abilities to damage enemies, with an implemented health and damage system. I also created melee, ranged, and magic enemies, each with their own behaviors inherited from a parent class. This project was primarily focused on learning new techniques and concepts in Unreal Engine.

Details

Engine: Unreal Engine
Genre: First person combat
Platform: PC (Windows)
